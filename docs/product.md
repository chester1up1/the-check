# Product Notes

## Vision

Let friends share music without caring which streaming platform each person uses.

## Primary user story

As a user, I want to paste a music link and instantly get equivalent links for other platforms, so I can share the right destination with friends.

## Core UX

1. User pastes a track link.
2. System detects the source platform.
3. System fetches or derives track metadata.
4. System finds candidate matches on other platforms.
5. UI shows best links with provider badges and confidence.

## MVP features

- input field for track URL
- provider detection
- cross-platform link results
- copy link action
- open provider action
- recent searches

## Later features

- playlists and albums
- shared rooms with friends
- group recommendations
- saved favorites
- auth and profiles

## Non-goals for MVP

- full social feed
- streaming playback inside the app
- complex moderation tools
